# Project Preferences

- H5 and web can use icons throughout the interface, especially bottom navigation, toolbars, cards, and action buttons.
- Preferred icon source: Iconfont project `http://iconfont.cn/users/18874/project?pid=66378` when its usable Symbol/CSS/Font Class asset link is available.
- Third-party UI component libraries are allowed when they reduce custom UI maintenance and fit the product style.
- If the Iconfont project page is not directly accessible, keep icons behind a local `Icon` component so the implementation can be swapped to Iconfont later without rewriting screens.
